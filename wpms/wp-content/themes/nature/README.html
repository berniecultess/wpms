Installation

How to install:

    1. Unpack this archive in your wp-content/themes/ directory.
    2. Go to Admin WordPress and select presentation.
    3. Select Nature 3.17 theme
    4. You need to do some customization to get it running.

Customization:

    - Alignment classes: by default all images are left aligned, remove this line to uncover it:
    #content p img{float:right;border:none;margin-left:5px;....
    - IMPORTANT
    - Open home php, find this line: Lorem ipsum dolor sit amet...
    change to your desired welcome text - Open home php, find this line:('cat=2&showposts=1')
    change cat=2 to your desired cat number, which can be found at your category admin page at left place: id
    - Stay at home php, find this line:('cat=3&showposts=1')
    change cat=3 to your desired cat number, which can be found at your category admin page at left place: id
    - Stay at home php, find this line:('cat=4&showposts=1')
    change cat=4 to your desired cat number, which can be found at your category admin page at left place: id
    - Stay at home php, find this line:('cat=1&showposts=1&offset=1')
    change cat=1 to your desired cat number, which can be found at your category admin page at left place: id
    - Open s footer php, find this line:('cat=5&showposts=5')
    change cat=5 to your desired cat number, which can be found at your category admin page at left place: id
    - You're done, have fun.

Usable Plugins:

    1. Ultimate Tag Warrior available at Neato
    2. WP Cache available at Uib
    3. Flickr RSS available at Eightface
